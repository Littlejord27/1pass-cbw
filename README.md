# 1pass-cbw
=====

## Installation

#### Mac OSX x64 install
```shell
mkdir ~/Documents/1pass-cbw
git clone https://github.com/jordan-cbw/1pass-cbw.git ~/Documents/1pass-cbw
cd ~/Documents/1pass-cbw
./install.sh mac
```


#### linux x32 install (Digital Ocean droplets)
```shell
mkdir ~/Documents/1pass-cbw
git clone https://github.com/jordan-cbw/1pass-cbw.git ~/Documents/1pass-cbw
cd ~/Documents/1pass-cbw
./install.sh linux
```
